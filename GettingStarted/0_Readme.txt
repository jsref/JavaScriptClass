An example of setting up a web page with Javascript.

These are best viewed via Dropbox on your local machine.  If you try
to view HowTo.html on the dropbox website the links won't work properly.

Visit the file HowTo.html in your browser to start.

The linked files (helloinline.html, helloexternal.html,
helloevent.html etc) are separate small examples of each approach
described in HowTo.html.  We are considering building a master example
out of these, but for the moment I thought they would be clearer
each illustrating a single example.
